Sistema Bancário
📄 Sobre o Projeto
Este é um sistema desenvolvido em Java focado na gestão de operações bancárias. O projeto utiliza uma estrutura modular para organizar o código-fonte e gerenciar as dependências do ambiente de desenvolvimento.

🚀 Funcionalidades

Gestão de Contas: Estrutura preparada para manipulação de dados financeiros e operações de entrada/saída.


Persistência Local: Organização de diretórios voltada para a compilação e armazenamento de binários.


Ambiente Padronizado: Configurações otimizadas para IDEs como IntelliJ IDEA, VS Code e Eclipse, garantindo que arquivos temporários não interfiram no versionamento.

🛠️ Tecnologias Utilizadas

Linguagem: Java.


IDE Sugerida: IntelliJ IDEA (configurações inclusas via arquivo .iml).


Versionamento: Git (com .gitignore configurado para múltiplas plataformas).

📁 Estrutura de Pastas

/src: Contém o código-fonte principal da aplicação.


.gitignore: Filtra arquivos desnecessários de IDEs (IntelliJ, Eclipse, NetBeans, VS Code) e do sistema operacional Mac OS.


Sistema_Bancario_atividade.iml: Arquivo de configuração de módulo para o IntelliJ IDEA.

⚙️ Como executar
Certifique-se de ter o JDK instalado em sua máquina.

Clone o repositório.

Importe o projeto em sua IDE de preferência (o arquivo .iml facilita a importação no IntelliJ).

Compile e execute a classe principal a partir do diretório
